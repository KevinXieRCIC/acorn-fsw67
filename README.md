# FSW67 计算器（GitHub Pages 一键部署）

## 部署步骤
1. GitHub 新建一个仓库（Repository）。
2. 上传本压缩包里的所有文件到仓库根目录（确保 `index.html` 在根目录）。
3. Settings → Pages → Source 选择 “Deploy from a branch”。
4. Branch 选择 `main`（或 `master`），Folder 选择 `/(root)`。
5. 保存后等待生成访问链接。

## 文件
- index.html：FSW67 计算器入口
- acorn_bc_favicon_pro.png：图标（可替换但请保持文件名）
